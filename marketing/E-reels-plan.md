# E Reels 規劃｜IG / FB Reels 內容策略

> Reels 跟 FB Ads 影片完全不同。本檔規劃帳號養成期 3 個月（90 支）內容。
> 主帳號目標：**單帳號破萬粉、月觸及 50 萬**。

## 0. Reels vs FB Ads 影片：兩個物種

| | FB Ads 影片 | Reels |
|---|---|---|
| 流量來源 | 付費投放 | 演算法自然推薦 + 推薦頁 |
| 連結 | 可放 CTA 按鈕 | **不能放外連結**（IG），CTA 靠 bio + 私訊 |
| 長度 | 30–90 秒 | **甜蜜點 15–30 秒**，最長 90 秒 |
| 製作度 | 可中高 | **越像素人越好**，高製作反而扣分 |
| 音樂 | 自選或購買 | **必須用 IG 推薦音樂**才會被推 |
| 字幕 | 加分 | **必加**（80% 靜音播放） |
| 發布頻率 | 一次性 | **每週 3–5 支，連續 3 個月**才養得起來 |
| 評估指標 | CPM、CPC、轉換 | 完播率、儲存、分享、追蹤 |
| 前幾秒 | 重要 | **第 1 秒定生死**，沒鉤住就滑掉 |
| 互動 | 留言、私訊 | **回覆每一則留言** = 演算法加分 |

→ Reels 是「**長期養帳號**」，FB Ads 是「**短期換轉換**」。兩條路要分開做。
