# F 工具評估｜把腳本變成實際成品

> 評估「你那端要裝什麼工具」+「我這端容器能直接幫你做什麼」+「外包/付費可選方案」。
> 結論：**圖文翻牌型 Reels（F2 格式）可以全程式化、零拍攝產出**；其他格式需要真人/實拍。
> 建議：先用零成本工具跑 1–2 個月，看數據再升級。

## 2. 「你那端」必裝工具清單（按優先級）

### 🔴 必裝（第一週）

#### A. CapCut 桌面版 + 手機版
- **用途**：核心剪輯工具，9 成 Reels 在這裡完成
- **費用**：免費（Pro 進階版 NT$120/月，**有需要再升級**）
- **下載**：https://www.capcut.com
- **為什麼**：自動字幕、IG 熱門音樂庫、模板多、上手快
- **替代**：Premiere（NT$700/月，太貴）/ DaVinci（免費但學習曲線陡）

#### B. Canva（網頁版）
- **用途**：圖文翻牌 Reels、IG 9 宮格、FB 圖卡、Story
- **費用**：免費 / Pro NT$400/月（**強烈建議 Pro**）
- **網址**：https://www.canva.com
- **為什麼**：模板量輾壓所有對手、有 Magic Resize、可一個圖匯出 9:16/1:1/16:9
- **替代**：Figma（免費，但模板少）

#### C. ManyChat（IG / FB 私訊自動回覆）
- **用途**：實現「私訊『新手』」CTA 系統
- **費用**：免費 / Pro USD$15/月
- **網址**：https://manychat.com
- **為什麼**：CTA 全靠這個跑、不裝 = Reels 流量接不住
- **替代**：Meta 內建自動回覆（陽春但夠用）、Chatfuel

#### D. Meta Business Suite
- **用途**：FB + IG 排程發佈、數據儀表板
- **費用**：免費
- **網址**：https://business.facebook.com
- **為什麼**：排程功能比第三方還順
- **替代**：Buffer（NT$200/月）、Later

### 🟡 建議裝（第二週起）

#### E. ElevenLabs（中文 TTS 配音）
- **用途**：旁白配音、不想真人錄音時用
- **費用**：免費 1 萬字/月 / Starter USD$5/月
- **網址**：https://elevenlabs.io
- **中文支援**：✅ 繁中流暢，可選女聲/男聲/年齡
- **為什麼**：F2 圖文翻牌可以全程式生成、不需員工出聲
- **替代**：OpenAI TTS（USD$15/百萬字元，更便宜）、剪映內建配音

#### F. Submagic 或 Captions.ai
- **用途**：自動上「動態字幕」（CapCut 字幕功能太陽春）
- **費用**：USD$10–25/月
- **網址**：https://submagic.co / https://captions.ai
- **為什麼**：IG Reels 字幕視覺直接影響完播率
- **替代**：CapCut Pro 字幕模板（NT$120/月，CP 較高）

#### G. Linktree 或 Beacons
- **用途**：IG bio 一個連結展開多個入口
- **費用**：免費 / Pro USD$5/月
- **網址**：https://linktr.ee / https://beacons.ai
- **為什麼**：IG bio 只能放 1 個連結，必須用這類聚合頁
- **替代**：自架 LIFF 入口頁

## 4. 「我這端」（容器內）可以直接幫你做的事

我這個容器是 Linux + Python + Node + Git，**可以裝 FFmpeg + Pillow + Playwright** 來自動化生成內容。

### ✅ 可以自動化生成的

#### A. F2 圖文翻牌型 Reels（**完全不需拍攝**）
- 從腳本讀取每一張卡的文字
- 用 Pillow / HTML 渲染成 1080×1920 圖
- FFmpeg 合成成 MP4，加轉場
- 加上 BGM（從本地音樂庫或下載）
- **產出**：MP4 檔，直接上傳 IG

範例：F2-1《新手 5 必做》——5 張翻牌、22 秒影片，可一鍵生成。

#### B. F5 對比型 split screen
- 程式化做左右分屏動畫
- 數字跳動效果（補間動畫）

#### C. 靜態圖卡（FB 動態 / IG Feed）
- 從文案批次生成圖卡
- 9 宮格自動排版
- 多比例匯出（1:1、9:16、16:9）

#### D. 字幕 SRT 檔
- 從口播逐字稿生成標準 SRT
- 自動斷句、時間軸

#### E. 配音 MP3
- 整合 ElevenLabs API（你需要提供 API key）
- 從腳本自動配音

#### F. 內容日曆 CSV
- 把 90 支 Reels 計畫匯出成 Excel/CSV
- 你直接匯入 Buffer / Meta Business Suite

### ❌ 我這端做不了的

- 真人拍攝（口播 F3、開箱 F4、倉庫 BTS F7）
- IG/FB 內建音樂庫（必須在 IG App 內選）
- 客戶 UGC（要真實徵集）
- 動態效果太花俏的剪輯（CapCut 模板更快）

## 6. 一鍵能做的：示範生成流程

我準備：

```
scripts/
├── generate_swipe_reel.py        # 生成 F2 圖文翻牌 Reels
├── generate_image_card.py        # 生成單張 IG 圖卡
├── generate_srt.py               # 生成字幕檔
├── generate_voice.py             # 呼叫 ElevenLabs API 配音
└── content_calendar.csv          # 90 支 Reels 排程匯出
```

只要：

```bash
python scripts/generate_swipe_reel.py reels/F2-1.json
```

就會吐出一支 1080×1920、22 秒的 MP4，可直接上傳 IG。
